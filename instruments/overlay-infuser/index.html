Overlay infuser

This script allows to put image overlay over the top of the developed HTML for comparison

Simply add this script in your html code and create folder 'overlays'. All images should be named overlay#(number should start from zero) and their file extension should be removed (i.e. overlay0, overlay1, overlay23). To see controls press question mark key while script is active

To use this script on mobile or tablet you will need an external keyboard
